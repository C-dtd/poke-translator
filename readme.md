# 로토무 번역기

- 포켓몬스터의 영문 버전 포켓몬 이름, 특성, 기술, 도구, 성격을 공식 번역된 단어로 바꾸는 크롬 확장프로그램입니다.
- 번역된 단어 위에 마우스를 올려두면 원문을 볼 수 있습니다.
- 오탈자나 잘못 매칭된 내용이 있을 수 있습니다.

## 크롬 확장프로그램 적용방법

1. chrome://extensions/ 에 접속합니다.
2. 오른쪽 상단의 개발자 모드를 활성화합니다.
3. 압축해제된 확장 프로그램 로드를 선택해 이 파일이 존재하는 폴더를 선택합니다.
4. 세부 정보에서 일반 확장 프로그램처럼 사용할 사이트 등을 설정할 수 있습니다.

## 로토무 번역기 설정

1. 활성화된 로토무 번역기 확장 프로그램을 클릭하면 로토무 번역기 설정창이 나옵니다.
2. 포켓몬 이름, 기술, 특성, 도구, 성격 등의 번역을 활성화/비활성화 할 수 있습니다.

## 주의사항

- 단어사전 형식으로 변환하는 방식으로 포켓몬 관련 문장이 아닌 경우에도 변환 될 수 있습니다. 전체 페이지에 적용보단 특정 사이트에서만 작동하게 설정하는 것을 추천합니다. (eg. bold => 대담)
